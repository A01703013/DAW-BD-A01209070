¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Es importante mencionar desde un principio que ambos lenguajes son totalmente diferentes, sin embargo pueden llegar a tener un par de aspectos en común, pero estos son mínimos. En primer lugar, JavaScript es un lenguaje interpretado, miestras que Java es un lenguaje compilado; dicho esto, los posible errores cometidos al programar en JavaScript son detectados al ejecutar el programa, mientras que con Java, los errores se detectan en tiempo de compilación. Otra gran diferencia es que Java es un lenguaje orientado a objetos, mientras que JavaScript es un leguaje orientado a prototipos. Java es un lenguaje tipado (cada variable tiene u tipo de datso en específico, mienstras que JavaScript es un leguaje debilmente tipado. En conclusión, JavaScript no es lo mismo que que Java, ni algún subconjunto de él [1].

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

Es importate mencionar que el objeto Date soporta métodos UTC, aunado a método de horarios locales [2].
  1. Date.now( ): devuelve el valor numérico correspondiente a la hora actual [2].
  2. Date.parse( ): transforma una cadena que representa una fecha y devuelve el número de milisegundo desde las 00:00:00 del 1 de enero de 1970, hora local [2].
  3. Date.prototype.getDate( ): devuelve el días del mes para la fecha especificada de acuerdo con a hora local [2].
  4. Date.prototype.getTime( ): devuelve el valor numérico correspondiente a la hora para la fecha especificada según la hora universal [2].
  5. Date.prototype.setDate( ): establece el día del objeto en relación con el comienzo del mes establecido actualmente [2].


¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

  1. Array.from( ): crea una nueva instancia de Array a partir de un objeto iterable [3].
  2. Array.isArray( ): determina si el valor pasado com argumento es un Array [3].
  3. Array.of( ): crea una nueva instancia Array con el número variable de elementos pasado como argumentos, independiente del número o del tipo [3].
  4. Array.prototype.concat( ): se usa para unir dos o más Arrays. Este método no cambia los Arrays existentes, más bien devuelve un nuevo Array [3].
  5. Array.prototypee.copyWithin( ): transfiere una copia plana de una sección a otra dentro del mismo Array, sin modificar su tamaño [3].


¿Cómo se declara una variable con alcance local dentro de una función?

Las varibales declaradas dentro de una función son varaible locales, para ello, se utiliza la instrucción let (o var) la cual declara una variable de alcance local, de tal manera que solo puede ser manipulada o accedida a ella dentro de la función en la cual fue declarada [4]

.
¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

El hecho de hacer uso de alguna variable local dentro de una función en específico genera que se modifique el valor de la variable global definifa con anterioridad. Por otro lado, si la variable global se define dentro de la función en cuestión, la variable es global y puede ser accedida desde cualquier otra función.La finalidad última de las variables globales es compartid datos entre las funciones de manera sencilla; sin embargo, se recomienda definir las variables como locales para que sean de uso exclusivo de la función en la cual fueron declaradas.
