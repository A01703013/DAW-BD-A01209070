Respuesta de las Preguntas

¿Qué es entonces una base de datos - DB?

Propiamente una base de datos es un conjunto de datos pertenecientes a un mismo contexto, los cuales son almacenados de una manera sistemática para poder usarlo posteriormente [1]. De esta manera, es una colección de archivos interrelacionados entre sí que tiene la última finalidad de proveer servicio a varios usuarios, de tal manera que se elimine la mayor redundancia posible y que sea independiente de la utilización que se le de [1].Es importante mencionar que las bases de datos deben de cumplir con ciertas características como la concurrencia, haciendo referencia a que múltiples usuarios pueden acceder simultáneamente a un fichero; seguridad, haciendo referencia al hecho de que deben de asegurar la protección de los datos frente a usuarios no autorizados; integridad, haciendo referencia a la validez y consistencia de los datos almacenados; y recuperación, haciendo referencia al protejo de los datos ante fallos en el sistema (copias de seguridad) [1].

¿En qué casos es conveniente usar bases de datos?

El principal caso en donde es conveniente el uso de bases de datos es en un sistema donde se manejen datos o archivos interrelacionados por diversos usuarios (clientes), como lo son hospitales escuelas, hospitales, bancos, empresas en general, por mencional algunos. De la misma manera, cabe mencionar que es de suam importancia tomar en cuanta el número (cantidad) de información a mnaejar en el sistema, ya que esto determiniaría el uso o no de un gestor de bases de datos [2].

¿Qué es un sistema de gestión de base de datos - DBMS?

Hace referencia a un conjunto de datos interrelacionados, así como a un conjunto de programas para acceder a ellos. De esta manera, permiten la manipulación, almacenamiento y consulta de los datos en los diversos ficheros.Es importante mencionar que el método de almacenamiento, así como el programa de gestión de los datos (DBMS) deben de ser independientes del programa cliente desde el cual se realizan las consultas. En otras palabras, se puede decir que un sistema gestor de bases de datos titne la finalidad de proporcionar un medio eficiente para la extración de datos en grandes bloques de información, a través de mecanismos muy óptimos [3].

Enlista y explica las funciones/responsabilidades que tienen los DBMS.

Los Sistemas de Gestión de Bases de Datos deben de tener 3 funciones principales: Función de definición: debe dar la posibilidad de describir los elementos de los datos, sus interrelaciones, estructuras y validaciones tanto a nivel externo como interno. Función de manipulación: debe permitir la busqueda, modificación, adición o eliminación de los datos. Función de utilización: debe permitir la carga de archivos, así como su la modificación de los registros, hacer copias de seguridad y proteger de usuarios no autorizados [4].

Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.
Considero que el proyecto en el que era viable la creación de una base de datos es para el Proyecto de Desarrollo de Videojuegos, mediante el cual sea posible el manejo de la información de una mayor cantidad de jugadores de ua manera más ordenada y sistemática. Dicho de esta manera la base de datos hubiera servido para llevar registros de los mejores puntajes, de mejoras sustanciales, eventos o de los niveles que tienen los jugadores o grupo de personas. De esta manera, hubiera sido posible la creación de un videojuego con mayores características como por ejemplo, un usaurio y contraseña de acceso por jugador, manejo del nivel de experiencia, o en su defecto almacenar todo el perfil de los jugadores, incluyendo logros y niveles jugados; entre muchos otros datos.
